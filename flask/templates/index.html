Explore How Artificial Intelligence and Code Interact

Machine Learning is made possible by increasing access to valuable data and compute. There has been a growing interest in the field to explore how AI may interface with source code to either refine existing code or augment the capabilities of a programmer. This webpage is a place to play with small-scale demos of programs inspired by this field of research. word2code uses the word2vec library to classify predetermined labeled code snippets with a natural language query.
